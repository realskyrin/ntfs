## 固定通知休眠

部分国产 ROM 或第三方系统移除了原生 Android 8.0 + 的固定通知休眠功能，于是就有了此应用，它可以休眠您系统通知栏中无法滑动消除的通知并在一段时间后自动恢复。快用它把你的通知栏清理的干干净净～～
